# DSA
数据结构和算法

### 快速排序
    快排的基本思想：分而治之
    主要步骤：
    1. 确定分界点，Arr[(left + right) >> 1] 或者 随机
    2. 调整范围：
           两个区间：第一个区间内的值都小于分界点的值
                     第二个区间内的值都大于分界点的值
    3. 递归处理左右两端
    
### 归并排序
    归并的基本思想：分而治之
    主要步骤：
    1. 确定分界点： mid = (left + right) >> 1;
    2. 递归排序 left right
    3. 归并 --> 合二为一
